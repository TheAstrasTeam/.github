## Why remake Scratch?

You know that Scratch is a **BEST** tool for new programmers and **PERFECT** toy for programmers.

But also you know that Scratch is SO FUCKING SLOW so that it cannot meet *certain demands* of programmers.

So we try to create a UNIQUE editer with JS/TS and RUST from zero, which rendering with WebGPU not WebGL.

And we can change the Bad Fucking Blockly from v9.0.0 to v13.0.0 and newer.

In my opinion, the best news is our new editor is compatible with Scratch Project File(.sb3)!
